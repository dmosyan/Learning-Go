Get a package <br />
`go get github.com/gorilla/mux`

List the packages <br />
`go list` <br />
`go list all` <br />
`go list -m all` <br />
`go list -m -versions github.com/gorilla/mux` - to check available version from the remote repo <br />

Verify dependency <br />
`go mod verify` - all modules verified
